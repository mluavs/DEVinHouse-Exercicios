Para treinar os conceitos de Spread.

    Crie dois arquivos:
  • index.html
  • app.js

Em app.js desenvolva uma função que receba dois arrays e realize a concatenação entre eles utilizando o operador Spread. Apresente o resultado no console.log(novoArray).

const novoArray = concatena([1, 2, 3], [4, 5, 6]);
console.log(novoArray);
// novo array deve ser [1, 2, 3, 4, 5, 6]
